Available Model Options

Data Generation Models

WhiteLightning supports all models available through OpenRouter, including:

  • openai/gpt-4
  • openai/gpt-4-turbo
  • anthropic/claude-3-opus
  • anthropic/claude-3-sonnet
  • mistral/mistral-large
  • google/gemini-pro
  • meta-llama/llama-2-70b

And many more! Check out the complete list at OpenRouter Models

To use any OpenRouter model, simply specify it with the --data-gen-model parameter:

$ python -m text_classifier.agent \
    -p "Your prompt here" \
    --data-gen-model "openai/gpt-4" \
    --model-type tensorflow

üí° Tip: You can use any model ID from OpenRouter. Just make sure to use the exact model ID as shown in the OpenRouter dashboard.

Command Line Usage

üöÄ Run models directly from command line

$ python -m text_classifier.agent \
    -p "Detect political bias in articles" \
    --generate-edge-cases true \
    --edge-case-volume-per-class 30 \
    --model-type tensorflow \
    --lang english \
    --max-features 5000

Train and deploy models with a single command. Supports various options for customization:

  • Model type selection (PyTorch, TensorFlow, scikit-learn)
  • Edge case generation
  • Language specification
  • Feature configuration
